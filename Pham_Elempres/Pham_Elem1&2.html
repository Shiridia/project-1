Navigation Tags


"The HTML nav element represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents. Common examples of navigation sections are menus, tables of contents, and indexes. This tag has full support from Chrome 6.0, Microsoft Edge 9.0, Firefox 4.0, Safari 5.0, and Opera 11.1." - MDN Web Docs / W3Schools

Anchor Tags


"The HTML a tag creates a hyperlink to other web pages, files, locations within the same page, email addresses, or any other URL. They can be in any form such as images, buttons, or text. This tag has full support from all browsers." - MDN Web Docs / W3Schools

Important Attributes

  • href
    Contains a URL or a URL fragment that the hyperlink points to.
  • target
    Specifies where to display the linked URL. Supplying "_blank" opens link in a new browser tab.
  • download
    This attribute instructs browsers to download a URL instead of navigating to it, so the user will be prompted to save it as a local file.
